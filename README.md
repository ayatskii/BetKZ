# BetKZ — Online Sportsbook Website

## Project Overview
BetKZ is a modern online sportsbook designed for users worldwide.

The project focuses on creating a fast, simple, and reliable betting website that works smoothly on both desktop and mobile devices.

Our goal is to replace slow, overloaded betting platforms with a clean and user-friendly experience.

## Why This Project Matters
Sports betting is popular all over the world, but many platforms still suffer from:

- Slow performance
- Complicated interfaces
- Long and unclear payout processes

BetKZ aims to solve these problems by offering:

- Quick access to sports events
- Clear betting options and odds
- A smooth and responsive website

The platform is designed for everyday users who want to place bets without unnecessary complexity.

## Target Users
- Sports fans aged 18–35
- Casual bettors interested in football, boxing, and e-sports

### User Needs
- Easy registration
- Fast website loading
- Clear bet tracking
- Mobile-friendly design

## Core Features (MVP)
**User flow:** Browse → Bet → Track → Withdraw

- **Account & Payments:** simple sign-up and balance management
- **Betting:** single bets, combined bets, early cash-out
- **User Dashboard:** bet history, balance overview, notifications
- **Admin Panel:** event management, odds setup, user monitoring

## Technology & Plan
- **Frontend:** React
- **Backend:** Go

### Timeline
- Weeks 1–4: user authentication and betting pages
- Weeks 5–8: payments and admin panel
- Week 9+: MVP launch
